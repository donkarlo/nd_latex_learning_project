    \section{Before running}
          The run engine must be on LuaLatex document class must not be standalone.
        
        \begin{itemize}
            \item In pycharm, right click on the page that contains the graph and from ``Modify run configuration`` change output directory from {{projectDir}/out} to {mainFileParent}.
            \item The graphname must not contain any underlines
            \item Two part words must be put in double quatations
        \end{itemize}
    \section{Simplest example}
